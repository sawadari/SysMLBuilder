# ケース05: 採掘フリゲートの運用ユースケース

## 背景
`UseCasesFrigate.sysml` からボトムアップに再構成した要求仕様である。

## ユースケース
### UC-MF-001 小惑星を採掘する
メインフロー:
1. 小惑星ターゲットを特定する。
2. 採掘レーザーを起動する。
3. 鉱石を採取し貨物倉へ格納する。
4. 貨物倉が満杯になったら採掘を中断する。

例外フロー:
- 採掘レーザーが故障した場合、採掘を停止してパイロットへ通知する。
- 対象小惑星が枯渇した場合、ターゲットを再取得する。

### UC-MF-002 鉱石を荷下ろしして補給する
メインフロー:
1. ステーションとのドッキング接続を確立する。
2. 鉱石をステーションへ移送する。
3. 重要システムを補給する。

例外フロー:
- ドッキングに失敗した場合、パイロットへ通知して荷下ろし手順を中止する。
- 貨物移送に失敗した場合、運用を中断して鉱石を船内に保持する。

## 記述方針
この入力では定量的な成功閾値を意図的に未指定としている。
