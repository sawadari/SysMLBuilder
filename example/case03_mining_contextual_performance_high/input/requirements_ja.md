# ケース03: 採掘フリゲートの文脈別性能要求

## 範囲
`MiningFrigateRequirementsDef.sysml` からボトムアップに再構成した要求仕様である。

## 要求
- REQ-MF-001: 採掘フリゲートの貨物容量は少なくとも5000 m3でなければならない。
- REQ-MF-002: 採掘フリゲートはHigh Sec運用において少なくとも200 DPSに耐えなければならない。
- REQ-MF-003: 採掘フリゲートはLow Sec、Null Sec、およびWormhole運用において少なくとも400 DPSに耐えなければならない。

## 設計コンテキスト
対象は複数の運用コンテキストで評価される採掘フリゲートである。
